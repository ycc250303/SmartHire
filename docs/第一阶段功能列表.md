
# SmartHire 第一阶段功能列表

记得写GET接口

## 用户注册登录模块 8081/user-auth

* 发送验证码
* 验证验证码
* 注册
* 登录

## 求职者数据模块 8082/seeker

* 获取当前求职者信息
* 更新求职者资料（在线简历）
  * 添加求职期望
  * 添加个人介绍
  * 添加实习/工作经历
  * 添加项目经历
  * 添加教育经历
  * 添加技能证书
* 上传附件简历
* 向岗位投递简历
  * 向HR发送消息和简历
* 基础筛选功能：根据求职期望内的字段筛选
  * 薪资
  * 工作地点
  * 岗位
  * 行业
  * 全职/实习/兼职
* 投诉岗位

## HR数据模块 8083/hr

* 获取HR信息
* 更新HR信息
* 岗位管理
  * 岗位发布
  * 岗位编辑
  * 岗位下线
  * 岗位列表查询
  * 岗位状态管理（招聘中、已结束）
* 简历管理
  * 简历接收与存储
  * 简历列表查看
  * 简历状态跟踪（未读/已读/通过/拒绝）
  * 简历搜索与筛选
* 基础匹配功能
  * 关键词匹配计算
  * 简历排序展示
* 投诉用户

## 管理员模块 8084/system

* 查询所有用户
* 查询所有岗位
* 封禁用户

## 共用功能

* 聊天功能
